# FaceIDApp

**FaceIDApp** is a simple Python project for face detection and recognition using a webcam.

## Features

- Real-time face detection
- Face recognition of registered users
- Easy to add new users

## Requirements

- Python 3.x
- OpenCV
- face_recognition
- NumPy
